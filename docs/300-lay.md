# Layouts & Panels

batipanel ships with 8 layout presets. Each one arranges panels differently to fit your screen size and workflow.

## Choosing a Layout

```bash
b myproject --layout 6panel     # one-time layout
b config layout 6panel          # change default
b layouts                       # list all available layouts
```

## Layout Overview

| Layout | Panels | Best For |
|--------|--------|----------|
| `7panel` (default) | 7 | AI coding + external monitor |
| `4panel` | 4 | Laptops (13-14") |
| `5panel` | 5 | Balanced workspace |
| `6panel` | 6 | General dev + large monitor |
| `7panel_log` | 7 | Full-width log bar |
| `8panel` | 8 | Dual Claude + monitoring |
| `dual-claude` | 7 | Multi AI agent + ultrawide |
| `devops` | 5 | Docker / K8s operations |

## Panel Types

Each panel runs a specific tool:

| Panel | Tool | Fallback |
|-------|------|----------|
| Claude | `claude` | Install prompt |
| Remote Control | `claude remote-control` | Install prompt |
| Monitor | `btop` | `htop` then `top` |
| Git | `lazygit` | `git status` |
| File Tree | `yazi` | `eza --tree` then `tree` |
| Docker | `lazydocker` | `docker ps` |
| Terminal | Plain shell | Always available |
| Logs | Plain shell | Always available |

## Creating Custom Layouts

1. Copy an existing layout: `cp ~/.batipanel/layouts/7panel.sh ~/.batipanel/layouts/myname.sh`
2. Edit the split commands and tool assignments
3. Test on both small and large terminals

See [Contributing](900-con.md) for details on submitting layouts upstream.
