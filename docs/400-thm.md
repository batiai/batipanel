# Themes & Appearance

batipanel includes 8 color themes that style the tmux status bar, window tabs, pane borders, and shell prompt.

## Changing Themes

```bash
b theme              # list all themes with current selection
b theme dracula      # apply a theme (takes effect immediately)
```

Themes persist across sessions — set it once, forget about it.

## Available Themes

| Theme | Style |
|-------|-------|
| `default` | Green/blue — clean and balanced |
| `dracula` | Purple/pink dark theme |
| `nord` | Arctic blue palette |
| `gruvbox` | Retro warm colors |
| `tokyo-night` | Blue/purple night theme |
| `catppuccin` | Pastel warm dark (Mocha) |
| `rose-pine` | Warm rose, soho vibes |
| `kanagawa` | Japanese ink painting palette |

## What Gets Themed

Themes apply to:

- **tmux status bar** — Powerline arrows, background, text colors
- **Window tabs** — active and inactive window indicators
- **Pane borders** — subtle color to distinguish panels
- **Shell prompt** — user, directory, git branch segments
- **Terminal colors** — background, foreground, cursor (via OSC sequences)

## Powerline Arrows

The status bar uses Powerline-style arrow separators. These require a Nerd Font. The installer handles this automatically:

- **macOS**: Installs MesloLGS NF via Homebrew and configures Terminal.app
- **Linux**: Installs to `~/.local/share/fonts` — you may need to select "MesloLGS NF" in your terminal preferences

If arrows show as `>` or `?`, your terminal font isn't set to a Nerd Font. See [FAQ](800-faq.md) for fixes.

## Font Setup

batipanel works best with a Nerd Font. The installer installs **MesloLGS NF** automatically.

For manual setup, download any Nerd Font from [nerdfonts.com](https://www.nerdfonts.com/) and set it in your terminal preferences.

## Color Depth

| Terminal | Color Support |
|----------|--------------|
| iTerm2 | True color (24-bit) — richest experience |
| Terminal.app | 256 colors — all themes work well |
| GNOME Terminal | True color |
| Windows Terminal | True color |
| Alacritty / Kitty / WezTerm | True color |

All themes are designed to look good in 256-color mode. True color terminals get slightly richer gradients.

## iTerm2 Native Integration

If you use iTerm2, batipanel can use native splits instead of tmux UI:

```bash
b config iterm-cc on     # enable (panes become iTerm2 splits)
b config iterm-cc off    # disable (use standard tmux + themes)
```

Note: In `-CC` mode, the tmux status bar and theme styling are not visible since iTerm2 manages the window chrome.

## NO_COLOR Support

batipanel respects the [NO_COLOR](https://no-color.org) standard. Set `NO_COLOR=1` or `BATIPANEL_NO_COLOR=1` to disable all color output:

```bash
NO_COLOR=1 b myproject
```
